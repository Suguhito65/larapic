# テーブル設計

## Users テーブル

| Column    | Type      | Options        |
| --------- | --------- | -------------- |
| id        | INT       | auto_increment |
| name      | VARCHAR   |                |
| email     | VARCHAR   |                |
| password  | VARCHAR   |                |
| create_at | TIMESTAMP |                |
| update_at | TIMESTAMP |                |

## Posts テーブル

| Column    | Type      | Options        |
| --------- | --------- | -------------- |
| id        | INT       | auto_increment |
| user_id   | INT       |                |
| body      | VARCHAR   |                |
| image_url | VARCHAR   |                |
| create_at | TIMESTAMP |                |
| update_at | TIMESTAMP |                |

## Comment テーブル

| Column  | Type    | Options |
| ------- | ------- | ------- |
| id      | INT     |         |
| comment | VARCHAR |         |
| post_id | INT     |         |
| user_id | INT     |         |

## Likes テーブル

| Column  | Type | Options |
| ------- | ---- | ------- |
| id      | INT  |         |
| post_id | INT  |         |
| user_id | INT  |         |

## Tags テーブル

| Column   | Type     | Options |
| -------- | -------- | ------- |
| id       | INT      |         |
| tag_name | VARCHAR  |         |

## Post_tag テーブル
